Hello from TEE

Verifiable Private RPC for Ethereum

About

An Ethereum RPC provider running inside Oasis ROFL with Intel TDX hardware security, providing private, verifiable access to Ethereum Mainnet.

The Goal: Part of the Kohaku Roadmap to provide private Ethereum RPC access. The server doesn't see which storage you're reading (using TEE + ORAM). Private queries, verified results.

Why This Exists

The Mission

Demonstrate Oasis ROFL as infrastructure for Ethereum's privacy initiatives. The SDK is pluggable, making this a viable private RPC option for wallets and applications.

Key Features

  • Trustless: Verifiable code execution - you can verify what code is running
  • Private: Server doesn't see what storage you're reading
  • Secure: Runs in Intel TDX trusted execution environment
  • Oblivious: ORAM-based private state reads (roadmap)

How to Connect

MetaMask Setup

Or add manually:

  1. Open MetaMask and ensure you're on Ethereum Mainnet
  2. Click the network dropdown and select Ethereum Mainnet
  3. Click on your account icon → Settings → Networks
  4. Find Ethereum Mainnet in the list and click Edit
  5. Under RPC URL, click "Add RPC URL" and enter:
    https://ethrpc.rofl.cloud
  6. Save and MetaMask will use the private RPC endpoint

Known Limitations

For Ethereum-native users, verifying the Oasis stack for trust establishment adds overhead compared to Ethereum-only solutions. This is a tradeoff for the privacy guarantees provided by the TEE architecture.
